Detailed Description

template<typename T, typename Allocator = allocator<T>>
class thrust::system::omp::vector< T, Allocator >

omp::vector is a container that supports random access to elements, constant time removal of elements at the end, and linear time insertion and removal of elements at the beginning or in the middle. The number of elements in a omp::vector may vary dynamically; memory management is automatic. The elements contained in an omp::vector reside in memory available to the omp system.

TThe element type of the omp::vector.
AllocatorThe allocator type of the omp::vector. Defaults to omp::allocator.
